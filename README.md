# git-practice

A beginner-friendly repository to learn and practice Git using the command line.

## Purpose

This repository is created to understand the basics of Git version control.  
It is meant for hands-on practice rather than production use.

## What You Will Learn

By working with this repository, you can practice:

- Initializing a Git repository
- Checking repository status
- Staging files
- Making commits
- Writing clear commit messages
- Viewing commit history
- Working with branches (basic level)
- Pushing code to a remote repository (GitHub)

## Basic Git Commands Covered

Some of the common commands you will use while practicing:

```bash
git init
git status
git add .
git commit -m "your message"
git log
git branch
git checkout
git pull
git push
